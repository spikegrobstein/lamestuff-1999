Toobur (pronounced too - ber) is a wysiwyg Hotline toolbar editor. It is about to be rewritten, so if you don't like something about it, please send a feature request to me by either emailing me or going to the comments section and filling out the forms.

For someone that knows anything about how Hotline toolbars work, the interface of Toobur is pretty straight forward, but for those of you who know nothing, I am working on documentation.

Toobur supports everything that is publicly known about Hotline toolbars: up, down, enter, leave sounds, connect icon, background, down, enter, disabled picts, ppats (import from resources), up/ down actions, and the show pict in window action. It comes bundled with 2 toolbars (neither of which were made using Toobur).

If you would like to get new toolbar to play around with, you can tool over to the official Hotline Hotline Server (hotline://hlserver.com) and go into the users area-> toolbars/soundsets -> toolbars section. In that folder, you should see an older version of Toobur and a couple hundred toolbars.

If you have any feature requests or bug reports, head on over to the Comments section of this website or email spike.
