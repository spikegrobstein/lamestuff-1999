The Flap class is a REALbasic 2.0 class that allows you to easily loop sounds and play Flaps.

The Flap file format was conceived and written by spike; inspired by an idea from his cousin. A Flap file is a file that is Mac only because it contains "snd " and "TEXT" resources that define how the song is played. The idea behind Flap is that if you listen to songs, you notice that it is really just made up of a bunch of loops- play one thing 4 times, play the next 2 times, then play the first one 8 times, etc. If you were to break the song up into the 6 or 7 loops that there are, the size is much smaller. What flap does is take the file that is filled with loops and plays them back in a specific order; a single Flap can contain several variations of the loops. Because of the fact that you can have any number of variations on a song, you can actually store many songs in a single file.

The Flap class does have some limitations though... First of all, it loads the entire file into RAM before playing, so if you have a single file with 20 songs in it, that can be a real waste of memory. Second, it doesn't have a couple of things in it that would be handy. When spike completes the Flap Class 2.0, it will have some extra functions and routines to further customize and give you much more control over how the Flap file actually works. It will also support embedding and saving Flaps and several other playing options and filtering options.

The Flap class comes with one example Flap (Smack My Bitch Up by Prodigy), the class, and an example that shows how to use the functions that are built in.
